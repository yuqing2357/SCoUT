# SCoUT

This repository is associated with a manuscript currently under peer review.

SCoUT is a curve-specific discrete representation framework for heterogeneous well-log archives. It is designed to learn curve-specific discrete tokens from multi-curve well-log data and to support downstream tasks such as lithology classification, porosity prediction, and geological semantic analysis.

## Code availability

The full source code, configuration files, trained model weights, curve-specific tokenizer codebooks, macro-token mapping files, and scripts required to reproduce the public-data experiments will be released upon acceptance of the manuscript.

During peer review, access to the private repository can be provided to editors and reviewers upon reasonable request.

## Data availability

The public datasets used in this study are available from their original sources, including NLOG, KGS, FORCE 2020, AGS DIG 2021-0027, and AGS SPE 006.

The private Jiyang Depression well-log dataset is subject to collaboration and data-governance restrictions and therefore cannot be publicly released.
